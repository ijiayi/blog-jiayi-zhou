Syntax

  1. Use :status: hidden to hide the article / page.
  2. Use :status: draft to craete draft page (under /drafts).
  3. Use <{filename}/cat/article2.rst> to link to content root.
  4. Use <{filename}cat/article2.rst> to link to current root.
  5. Using {tag}tagname and {category}foobar to link to tags and categories.
